### BrokenBlockingQueue.java (module9-debug)

```java
/**
 * ğŸ”´ BROKEN CODE - FIND THE BUGS!
 * Expected bugs: 3 | Time target: < 5 minutes

public class BrokenBlockingQueue<E> {
    
    private final Queue<E> queue = new LinkedList<>();
    private final int capacity;
    
    public synchronized void put(E element) throws InterruptedException {
        // BUG #1: Using 'if' instead of 'while' - spurious wakeup!
        if (queue.size() == capacity) {
            wait();
        }
        queue.add(element);
        
        // BUG #2: Using notify() instead of notifyAll()
        notify();
    }
    
    public synchronized E take() throws InterruptedException {
        if (queue.isEmpty()) {
            wait();
        }
        E element = queue.poll();
        notify();
        // BUG #3: Not checking if poll() returned null
        return element;
    }
}
```
